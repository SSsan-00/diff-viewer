# diff-viewer 利用マニュアル

## 1. このツールでできること
- 左右2つのコードを並べて差分を確認できます。
- 複数ファイルをまとめて読み込んで比較できます（ファイル単位の行番号やジャンプが可能）。

## 2. 起動方法
- `index.html` を **file:// 直開き**で利用できます（サーバ不要）。

## 3. 基本的な使い方
### 左右ペインの役割
- 左ペインに比較元、右ペインに比較先を読み込みます。

### コードの貼り付け
- 左右のペインに直接貼り付けても比較できます。

### ファイル読み込み（複数ファイル対応）
- 各ペインの「ファイルを選択」から読み込みます。
- 複数ファイルを選ぶと、ペイン内でファイルごとに連結表示されます。
- 読み込んだファイルは **ファイルカード**として横並びに表示されます。

### ファイルカードの使い方
- カードをクリックすると **該当ファイルの先頭**へジャンプします。

## 4. 差分の見方
### 行レベル差分
- 追加・削除・置換の行が色分け表示されます。

### 行内差分（inline diff）
- 行の中の「変化した部分」だけが強調されます。
- インデント差・空白差も行内差分として見えます。

## 5. アンカー機能の使い方
- アンカーは「左右の対応行を固定する」ための目印です。
- 行番号領域をクリックしてアンカーを付けられます。
- アンカー一覧から該当行へジャンプできます。
- 同じ行をクリックするとアンカーは解除されます。

## 6. 便利な操作
### 行ジャンプ（Ctrl+g）
- **ファイル単位の行番号**でジャンプできます。
- デフォルト対象は **現在フォーカスしているファイル**です。
- ファイル選択UIで対象ファイルを切り替えられます。

### 検索（Ctrl/Cmd+F）
- フォーカス中のペインに検索UIを開きます。

### テーマ切り替え（ライト / ダーク）
- ヘッダーのテーマトグルで切り替えられます。

### シンタックスハイライト
- 「ハイライト」トグルで ON/OFF を切り替えます。

### 差分のみ表示
- 「差分のみ表示」トグルで、差分の無い行を折りたたみ表示にします。

## 7. よくある疑問・注意点
- ハイライトが見づらい場合は、テーマ切り替えやハイライトON/OFFを試してください。
